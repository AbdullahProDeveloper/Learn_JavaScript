গুরুত্বপূর্ণ পরামর্শ

আপনার PDF অর্ডার সিস্টেমের জন্য যেহেতু ইউজার অ্যাকাউন্ট এবং ডাটাবেজ প্রয়োজন, তাই Netlify বা Firebase Hosting বেছে নিন। GitHub Pages শুধুমাত্র static ওয়েবসাইটের জন্য suitable।

Netlify তে Deployment করার

  1. Netlify.com এ গিয়ে সাইন আপ করুন
  2. "New site from Git" নির্বাচন করুন
  3. আপনার GitHub repository connect করুন
  4. Build settings configure করুন
  5. Deploy করুন এবং আপনার site link copy করুন
  6. কাস্টম ডোমেইন add করুন (যদি প্রয়োজন হয়)
বিস্তারিত指南

Firebase Hosting এ Deployment

  1. Google অ্যাকাউন্ট দিয়ে Firebase এ login করুন
  2. একটি new project তৈরি করুন
  3. Firebase CLI install করুন
  4. আপনার প্রোজেক্ট ফোল্ডারে initialize করুন
  5. ফাইলগুলি deploy করুন
  6. আপনার hosting URL টি share করুন

সীমাবদ্ধতা সম্পর্কে

ফ্রী হোস্টিং options গুলোতে কিছু সীমাবদ্ধতা থাকে। যদি আপনার ওয়েবসাইটে অনেক traffic আসে, বা বেশি storage প্রয়োজন হয়, তাহলে paid plan এ upgrade করতে হতে পারে। তবে শুরু করার জন্য ফ্রী optionsই যথেষ্ট।
